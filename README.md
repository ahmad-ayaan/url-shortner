# 🔗 URL Shortener

A simple URL Shortener application built using **Spring Boot** following the **MVC (Model-View-Controller) architecture**. This project allows users to convert long URLs into shorter, easy-to-share links and redirect users to the original URL using the generated short code.

---

## 🚀 Features

- Generate short URLs from long URLs
- Redirect short URLs to the original URL
- MVC architecture implementation
- RESTful API design
- URL mapping management
- Database persistence
- Clean separation of Controller, Service, and Repository layers

---
